SVG

Aus SELFHTML-Wiki

Das auf XML basierende SVG (Scalable Vector Graphics) ist ein vom W3C empfohlenes Grafikformat zur Beschreibung zweidimensionaler Vektorgrafiken.

[Bearbeiten] Vorteile

  • präzise Beschreibung von Vektorformen
    SVG ist das einzige im Web breit einsetzbare Vektorgrafik-Format. Man sollte es einsetzen, wenn sich die meisten Bildelemente mit SVG-Formen ausdrücken lassen und eine clientseitige Skalierbarkeit oder eine Dynamik gewünscht ist. Letzteres ist in Zeiten von unterschiedlichen Device-Pixel-Ratios so gut wie immer der Fall.
    Darunter fallen eine große Zahl von Grafiken im Web: Icons, Logos, Illustrationen/Animationen, manche Zeichnungen, komplexe UI-Elemente, Karten, Diagramme/Visualisierungen usw. Für die wurden bisher Rastergrafiken, Flash-Plugins, HTML5-Canvas, Icon-Fonts usw. verwendet. SVG ersetzt diese Anwendungen zunehmend, aus praktischen Gründen aber nicht alle.
  • Animationen, Transformationen
    Besonders im Bereich interaktiver Grafiken und der Datenvisualisierung werden SVG-Grafiken nahtlos in HTML-Dokumente eingebunden und über JavaScript programmiert.
  • Formatierung mit CSS-Techniken
  • DOM-Scripting
    Wie HTML kann der SVG-Code durch JavaScript interaktiv erzeugt und verändert werden.

[Bearbeiten] Einbindung

Die Einbindung von .svg- Grafiken ist sehr einfach, entweder direkt über das img-Element, oder mit Hilfe von object. Diese Variante bietet folgende Vorteile:

  • Fallback-Text oder .png-Bild für den IE8.
  • Animation des Objekts durch JavaScript möglich
  • Formen können im SVG-Quelltext mit Links ergänzt werden
Beispiel: svg-Grafiken einbinden
<!doctype html> <html> <head> <meta charset="utf-8"> </head> <body> <h1>svg-Grafiken einbinden</h1> <img alt="Stern Namibia" src="Stern Namibia Flagge.svg"> <object data="Stern Namibia Flagge.svg" type="image/svg+xml"> <!---Fallback---> Ihr Browser kann leider kein svg! </object> </body> </html>

SVG-Grafiken können aber auch als background-image eingebunden werden.

Alternativ kann eine SVG-Grafik mithilfe des svg-Elements inline im HTML dargestellt werden. Dieser Code kann dann durch CSS formatiert, bzw. durch JavaScript animiert und transformiert werden.
