[Bearbeiten] If - Anweisung in PHP

In PHP kann man mit der sogenannten If-Anweisung auch Kontrollstruktur genannt, Variablen prüfen. Man überprüft eine Bedingung, ist sie wahr dann soll was passieren und das gleiche beim Falschen auch.

Beispiel 
$a = 10;
$b = 20;
if($a > $b) {
  print "Einkaufen gehen";
}
else {
  print "Nicht einkaufen gehen";
}

In diesem Beispiel haben wir zwei Variablen, eine mit dem Wert 10 und eine mit 20. Die If-Anweisung überprüft ob $a größer $b ist. Ist das der Fall dann schreibt PHP »Einkaufen gehen«. Um jetzt keine zweite If-Anweisung schreiben zu müssen die den gegenteiligen Fall abdeckt gibt es die else-Anweisung. Wenn also $a kleiner $b ist dann schreibt PHP »Nicht einkaufen gehen«.

If heißt übersetzt: Wenn (im Falle von als). In unserem Beispiel also: Wenn $a größer als $b dann schreibe »Einkaufen gehen« ansonsten schreibe »Nicht einkaufen gehen«.

[Bearbeiten] Passwortbeispiel

Wir legen zwei neue Dateien an, eingabe.html und ausgabe.php:

Beispiel: eingabe.html 
<form action="ausgabe.php" method="post">
 <input type="password" name="pw">
 <input type="submit" value="Senden">
</form>

Wir erstellen ein neues Formular das an ausgabe.php versendet wird. In das Passwortfeld müssen wir später unser Passwort einschreiben und der submit-Button bedeutet das das Formular jetzt abgesendet wird.

Beispiel: ausgabe.php 
if($_POST["pw"] == "test") {
  print "Herzlich Willkommen im geschützten Bereich";
}
else {
  print "Falsches Passwort";
}

Per If-Anweisung überprüfen wir ob das eingegebene Passwort dem Wert »test« entspricht, wenn ja dann hast du Zutritt zum geschützten Bereich, ansonsten nicht.
