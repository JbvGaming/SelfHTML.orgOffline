CSS/Eigenschaften/Animation/Transition

Aus SELFHTML-Wiki
< CSS | Eigenschaften | Animation(Weitergeleitet von Transition)
Wechseln zu: Navigation, Suche

Durch transition können Sie Wertänderungen von CSS-Eigenschaften animieren. Während eine Änderung mit :hover bis jetzt nach dem Motto „Ein/Aus“ umgeschaltet wurde, kann man mit transition die Werte allmählich oder schrittweise ändern.

Beispiel: Navigationsleisten mit einem weichen Übergang einblenden ansehen …
nav ul li ul {
  margin: 0; 
  padding: 0;
  position: absolute;
  top: 2.5em; 
  font: 0/0 serif;            /* Unternavigation ausblenden */
  z-index: -1;
  -webkit-transition: all 0.5s;
  transition: all 0.5s;
}
 
nav ul li:hover ul {
  font: inherit;
  z-index: auto;               /*  Unternavigation einblenden */
}
ansehen …

Im Beispiel werden Navigationselemente, die von der Maus überfahren werden, vergößert, wobei dies allmählich innerhalb von 0.5 Sekunden passiert. Die Untermenüs werden ebenfalls mit einem weichen Übergang eingeblendet. Der Hintergrund der Navigationselemente wurde mit linear-gradient()-Verläufen gestaltet.


Folgende Angaben sind möglich:

  • none:
  • transition-property: legt fest, welches Element verändert werden soll
  • duration: legt die Dauer der Veränderung fest
  • timing-function(): definiert die Geschwindigkeitskurve des Transitions-Übergangs
  • delay(): Verzögerung
  • initial: Werte werden auf Standardwert zurückgesetzt
  • inherit: Werte werden vom Elternelement übernommen

[Bearbeiten] transition-property

Die Eigenschaft transition-property legt fest, welches Element verändert werden soll.

Folgende Angaben sind möglich:

  • none: Die transition wird nicht ausgeführt
  • all : Alle Eigenschaften werden angesprochen (Standardwert)
  • Eigenschaft: durch Komma getrennte Liste von CSS-Eigenschaften
  • initial: Werte werden auf Standardwert zurückgesetzt
  • inherit: Werte werden vom Elternelement übernommen

[Bearbeiten] duration

Die Eigenschaft transition-duration legt die Dauer der Veränderung fest.

Folgende Angaben sind möglich:

  • Zeit: Zeitangabe in Sekunden(s) oder Millisekunden(ms)
  • initial: Werte werden auf Standardwert zurückgesetzt
  • inherit: Werte werden vom Elternelement übernommen

[Bearbeiten] timing-function

Die Eigenschaft timing-function() definiert die Geschwindigkeitskurve des Transitions-Übergangs.

Beispiel ansehen …
.linear .auto {
  transition-timing-function: linear;
}
 
.ease-in .auto {
  transition-timing-function: ease-in;
}
 
.ease-out .auto {
  transition-timing-function: ease-out;
}
 
.ease-in-out .auto {
   transition-timing-function: ease-in-out;
}
ansehen …

Im Beispiel sehen sie, wie die Animation je nach gewählter timing-function unterschiedlich beschleunigt. Allerdings kommen alle gleichzeitig im Ziel an. Die Animation wird mit der Pseudoklasse :target ausgelöst.

Übrigens, es wurden keine externen Bilder eingebunden, die Grafiken wurden alle mit CSS erzeugt. Die Farbwerte wurden mit der CSS-Variable currentColor festgelegt, so dass spätere Änderungen nur im Farbwert des html-Elements getätigt werden müssen.

Folgende Angaben sind möglich:

  • ease: Übergang mit langsamen Start, beschleunig und endet langsam (Standardwert) (⇔ cubic-bezier(0.25,0.1,0.25,1))
  • linear: Übergang mit gleicher Geschwindigkeit vom Start bis zum Stop (⇔ cubic-bezier(0,0,1,1))
  • ease-in: Übergang mit einem langsamen Start (⇔ cubic-bezier(0.42,0,1,1))
  • ease-out: Übergang mit einem langsamen Ende (⇔cubic-bezier(0,0,0.58,1))
  • ease-in-out: Übergang mit einem langsamen Start und Ende (⇔ cubic-bezier(0.42,0,0.58,1))
  • cubic-bezier(n,n,n,n): möglich sind numerische Werte zwischen 0 bis 1
  • initial: Werte werden auf Standardwert zurückgesetzt
  • inherit: Werte werden vom Elternelement übernommen

[Bearbeiten] delay

Die Eigenschaft delay() legt fest, mit welcher Verzögerung der Übergang ausgelöst werden soll.

Beispiel ansehen …
figure h2 {
  transform: translateY(-200px);
  transition: all 0.3s ease-in-out;
}
 
figure:hover h2 {
  transform: translateY(0px);
  transition-delay: 0.4s;
}
 
figure p {
  transform: translateX(200px) rotate(90deg);
  transition: all 0.4s ease-in-out;
}
 
figure:hover p {
  transform: translateX(0px) rotate(0deg);
  transition-delay: 0.2s;
}
 
figure a.info {
  transform: translateY(-200px);
  transition: all 0.2s ease-in-out;
}
 
figure:hover a.info {
  transform: translateY(0px);
  transition-delay: 0.3s;
}
ansehen …

Im Beispiel werden die Bildunterschriften bei :hover mit unterschiedlichen Verzögerungen eingeblendet. Für die Einblendung werden die verschiedenen Funktionen von transform verwendet.

Folgende Angaben sind möglich:

  • Zeit: Zeitangabe in Sekunden (s) oder Millisekunden (ms)
  • initial: Werte werden auf Standardwert zurückgesetzt
  • inherit: Werte werden vom Elternelement übernommen

[Bearbeiten] Browser-Support

Achtung!

Die Eigenschaft transition ist derzeit (September 2014) nur in die Browser Android 4.4, Chrome, Firefox, Internet Explorer 10+, Safari 5.2 und Opera implementiert, deshalb muss man proprietäre Eigenschaften verwenden.

für Android 2.3-4.3, und Safari 5.1:

  • -webkit-transition

Hinweis:

Sie sollten bei der Definition solcher noch nicht umfassend verbreiteter Eigenschaften die herstellerspezifischen Angaben immer vor der standardisierten Form machen. Da später gemachte Angaben frühere überschreiben, ist so sichergestellt, dass die standardisierten Angaben von dem Zeitpunkt an, an dem sie unterstützt werden, auch verwendet werden.
