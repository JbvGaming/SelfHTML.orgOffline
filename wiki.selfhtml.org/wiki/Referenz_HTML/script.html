Beachten Sie

Das immer noch oft genutzte Attribut language wurde als deprecated eingestuft.

Tipp

In HTML5 ist der Defaultwert von typetext/javascript. Das bedeutet, Sie brauchen nur dann ein type-Attribut anzugeben, wenn Sie eine andere Scriptsprache als JavaScript verwenden.
