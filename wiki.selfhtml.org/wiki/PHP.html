[Bearbeiten] Übersicht

PHP ist eine serverseitige Programmiersprache mit der sich dynamische Webseiten und Inhalte erzeugen lassen. Das heißt, dass die Webseite kein fertiges Dokument ist, sondern erst auf Anforderung durch den Client (z.B. Browser), einen Request also, zusammengestellt wird. Dazu wird der PHP-Code zunächst auf dem Server ausgeführt und die Ausgabe an den Client gesendet. Man unterscheidet in Websites häufig zwischen PHP in HTML und HTML in PHP, wobei letzteres zwar mehr Möglichkeiten bietet, aber wesentlich komplizierter ist.

Grundlegende Anwendungsbeispiele für PHP sind:

  • Verarbeitung von Formularen: Eingaben auswerten und Daten speichern bzw. per E-Mail an den Seitenbetreiber versenden
  • Login-Möglichkeit anbieten: Registrierung ermöglichen, Daten verschlüsseln, Daten in einer Datenbank speichern
  • Zusammensetzung verschiedener Dateien: Daten in Template einfügen
Beispiel: Codebeispiel für PHP 
<?php
  echo 'Hello World!';
?>
// oder
<?php $myHello = "Hello World" ?>
//oder
<p>Und mitten im HTML: <?php echo $myHello ?></p>
