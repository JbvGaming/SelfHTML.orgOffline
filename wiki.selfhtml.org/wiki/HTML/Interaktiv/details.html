Das details-Element ermöglicht es Seiteninhalte solange zu verstecken, bis es durch einen Klick auf summary aufgeklappt wird.

  • 27
  • 6.1
  • 22

Details: caniuse.com

Beispiel ansehen …
<!doctype html> <html> <head> <meta charset=utf-8> <title>Details</title> </head> <body> <details> <summary> Update News: </summary> Das neue <details>-Element ermöglicht es, ganz ohne den Einsatz von JavaScript, ursprünglich verborgene Textinhalte aufzuklappen. </details> </body> </html>
ansehen …

details bezeichnet einen semantischen Abschnitt, der mit Text, Bildern (oder einem figure-Element, einem Formular oder auch anderen details) gefüllt werden kann.

Ein Klick mit der Maus öffnet den versteckten Abschnitt. Alternativ bringt die Tab-Taste die Überschrift summary in den Fokus. Ein Klick auf Leertaste oder Enter öffnet dann den Text.

[Bearbeiten] Das Summary-Element

Mit summary können Sie dem Akkordeon eine Überschrift geben.

Beachten Sie: Wenn kein Summary-Element verwendet wird, zeigt der Browser das Dreieck mit dem Text Details an.

[Bearbeiten] open

Mit dem Klick auf die summary wird details das Attribut open angehängt. Dies können Sie sowohl direkt in den HTML-Code schreiben, als auch mit CSS stylen:

Beispiel: Einbindung von open 
<details open> 
  <summary>
    Browser-News:
  </summary>
  Während Chrome und Safari das neue Element schon länger unterstützen, 
  ist es jetzt auch in Opera 22 verfügbar!
</details>


Beispiel: CSS-Formatierung für open 
details[open] {
  background: coral;
  margin-bottom: 1em;
}

[Bearbeiten] CSS-Formatierung des details-markers

  • 27
  • 6.1
  • 22

Details: caniuse.com

Sie können den details-marker (das kleine schwarze Dreieck) in begrenztem Umfang formatieren[1]. Allerdings gibt es hierfür noch keinen Standard, so dass es in manchen Browsern doch anders angezeigt wird.

Da die Form des details-markers nicht direkt geändert werden kann, wird er mit display:none ausgeblendet und stattdessen ein Pseudoelement erzeugt und formatiert:

Beispiel: CSS-Formatierung des details-Symbols ansehen …
summary::-webkit-details-marker {
  display: none;
}
summary:before {
  content: "+"; /* Verwendung des "plus"-Symbols anstelle des Dreiecks */
  color: green;
  margin-right: 5px;
}
ansehen …

[Bearbeiten] Browserunterstützung

Diese Formatierung mit -webkit-details-markerspricht nur die Webkit-Browser Chrome, Safari und Opera an.

Allerdings unterstützen der IE und Firefox details derzeit nicht.[2]. Für diese Browser gibt es aber ein jQuery-Polyfill[3].
